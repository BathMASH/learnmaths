Some software you might find useful

    Concept Maps
  • Coggle has LaTeX support for mathematical notation
  • FreeMind
  • FreePlane - based on FreeMind, but with support for LaTeX if you want to use mathematical notation
    Drawing Graphs and other visualisation
  • GeoGebra - also has a 'slider' function so you can investigate what happens if you change a variable
    Typing Notes
  • It is possible to type mathematical notes in Microsoft Word including entering equations without using the mouse. Information to get you started: Getting started with effective entry of equations in Word.
  • To use more advanced mathematical notation and diagrams some people decide to learn to use LaTeX, a typesetting system for mathematical documents. Information to get you started: Getting started with LaTeX.
    Audio software
  • Audacity
  • NCH Wavepad
