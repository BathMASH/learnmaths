Thinking Styles

So how do you think about maths?

This test helps you investigate how you think when learning. It measures four scales:
Active/Reflective; Sensing/Intuitive; Visual/Non-visual; Global/Sequential
Details on each scale can be read on a separate page, or viewed with the results of the test.

In each section there will be several questions, each with two options.

We need to find out how you would act in a range of different situations. Even if a question doesn't seem to fit, try to answer it as accurately as you can.
