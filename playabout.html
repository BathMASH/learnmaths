Using examples

Look for examples in the notes or a recommended textbook and show how it relates to the definition. For some definitions it will be easy to come up with diagrams, for others it will be harder. If you can draw a diagram, try to use it to illustrate all the key points.

For the example diagram below showing { 1 / n } n = 1 inf , identify what various parts of the graph show.

The real sequence \(\{\)\(a_n\)\(\}_{n=1}^{\inf}\) converges to the real limit \(l\) if for every \(\varepsilon\) \(> 0\) there exists an \(N\) \(\in\) \(\mathbb{N}\) such that for every \(n \ge N\) \(|a_n - l|< \varepsilon\).
